## 串数据类型的定义

- 串（string）的定义：由零个或者多个字符组成的有限序列

  ```c
  char str[] = "abcdef";
  ```

- 串的存储结构

  ```c
  /* 定长顺序存储表示 */
  typedef struct {
      char str[maxSize+1]; /* maxSize 为已经定义的常量 +1 是因为多出一个 \0 作为结束标记 */
      int length;
  }

  /* 变长分配存储表示 */
  typedef struct {
      char *ch;  /* 指向动态分配存储区首地址的字符指针 */
      int length;  /* 串长度 */
  }
  ```
